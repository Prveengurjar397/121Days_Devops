# Day 04 – Linux Text Editors (Vim & Nano)

## 🎯 Task

Today's goal is to learn how to create, edit, save, and manage files using Linux text editors.

Text editors are essential tools for every Linux Administrator and DevOps Engineer because configuration files, scripts, YAML files, Dockerfiles, and Kubernetes manifests are all edited from the terminal.
