# Day 05 – Linux User & Group Management

## 🎯 Task

Today's goal is to understand how Linux manages users and groups.

Every Linux system can have multiple users. As a DevOps Engineer, you must know how to create users, manage groups, assign permissions, and control access to the system.

- Delete User (`userdel`)
- Create Group (`groupadd`)
- Delete Group (`groupdel`)
- Add User to Group (`usermod -aG`)
